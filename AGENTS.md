This project is Tachikoma — an OpenCode agent plugin.

- **Source of truth**: `src/` — all development happens here.
- **Runtime install**: `.opencode/` — created by `bun run install.ts`, which copies `src/` into the correct OpenCode directories. Never edit `.opencode/` directly.
- **Docs**: https://opencode.ai/docs — reference for skills, tools, agents, and plugin APIs.
